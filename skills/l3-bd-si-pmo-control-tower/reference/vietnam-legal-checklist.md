# Vietnam Legal & Contracting Checklist — KT × VNPT (체결 전 점검표)

> ⚠️ 일반 정보이며 법률·세무 자문이 아니다. **체결 전 베트남 자격 변호사·세무사 검토 필수**, 심층 조항 검토는 `L5-bd-contract-review`(준거법 베트남)로 위임한다. 모든 법령·세율·기관은 변동 가능 — **체결 시점 재확인**.

PMO가 본계약/라이선스 단계 진입 시 ⑩(규제)·⑫(계약·IP)에서 본 표를 점검한다.

## 1. 세금 — 외국계약자세(FCT)
- [ ] KT 지급액에 **VAT+CIT 원천징수** 적용(VNPT가 원천징수·납부) 전제 확인
- [ ] **분류** 확정: 기기=goods(저세율) / SW 라이선스=**royalty(CIT 10%)** / 서비스 / 혼합(기기+설치) — 분류가 세율·마진을 좌우
- [ ] 디지털서비스 VAT **5→10%**(Circular 69/2025) 반영
- [ ] **gross-up** 조항(세후 수취액 보장 여부) 명문화
- [ ] **한–베 조세조약(DTA)** 적용으로 로열티·CIT 경감 가능성 — 적용 요건·서류 확인
- 근거: Circular 103/2014·69/2025, Decree 122/2024. 체결 전 세무사 확인.

## 2. 데이터 보호 — PDPL 2025 / Decree 356
- [ ] 적용 법령 = **개인정보보호법(PDPL, 법 91/2025/QH15, 2026.1.1 시행)·Decree 356/2025** (구 Decree 13/2023 대체)
- [ ] 데이터 **VNPT Cloud(베트남 내) 거주** 설계
- [ ] **DPIA**(처리영향평가) + **CTIA**(국외이전영향평가) 수행 — 국외 이전(예: AI 클라우드 처리) 발생 시
- [ ] **컨트롤러/프로세서 계약**(역할·의무) 체결
- [ ] **DPO**(데이터보호책임자/조직) 지정
- [ ] 국외이전 위반 과징금 **최대 직전연도 매출 5%** 인지
- 근거: Law 91/2025/QH15, Decree 356/2025. RFI #7 필수 요건.

## 3. 기술이전 — Law on Technology Transfer 2017
- [ ] KT SW/기술 라이선스가 **기술이전** 해당 여부 검토
- [ ] 해당 시 **MOST(과학기술부) 등록** 필요성 확인(세제·집행에 영향)
- [ ] 기술이전 계약 **베트남어** 작성(라이선스 dossier 요건)

## 4. 계약 언어·형식
- [ ] 일반계약: **영-베 이중언어 + 우선본(prevailing) 조항**
- [ ] 기술이전·IT·소비자 영역은 **베트남어** 요구 가능
- [ ] 법원·중재·공증 제출 시 **베트남어 번역**(공증)
- [ ] 실행 형식: **법적대표자 서명 + 회사 직인(con dấu)** — 유효성 실무 확인
- 근거: 민법 2015·상법 2005, 기술이전법 2017, 공증법.

## 5. 분쟁해결
- [ ] 기관 선택: **VIAC**(베트남, 집행 신속) vs **SIAC/ICC**(중립, 단 외국판정은 베트남 법원 **승인** 필요)
- [ ] 베트남은 **뉴욕협약** 가입(170여 개국 집행)
- [ ] **국영기업(VNPT) 주권면제 배제** — 중재 합의 시 일반 법인과 동일 집행
- [ ] 중재조항 **서면 + 인정된 기관 명시**(미특정 시 무효 사례 존재)
- [ ] 외국요소로 **중재 언어 영어** 합의 가능
- [ ] '베트남 법 기본원칙 위반' 거부사유(모호) 리스크 인지 → 조항 정밀 작성
- 근거: 상사중재법(LCA) 2010, 민사소송법 2015 Part 7, 뉴욕협약.

## 6. 거래상대(SOE)·기타
- [ ] VNPT **국영그룹 내부 승인 체인** → 일정 버퍼 반영
- [ ] **통화·환·외환관리** 조항(지급 통화·환리스크)
- [ ] **인증(MIC ICT·Google Cert)** 책임·일정 게이트·비용 분담
- [ ] 통신/클라우드 관련: 통신법 2023·Decree 163(라이선스·통지) 적용 여부

## 판정
- 모든 항목 ✅ → 본계약 진행 / 미충족 → `L5-bd-contract-review` 심층 검토 후 보완·재점검.
